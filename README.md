# 📊 Swiggy Sales Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **Excel dashboard** built to analyze Swiggy sales data. It provides insights into sales trends, customer behavior, and performance metrics.

---

## 🎯 Objectives

* Analyze total sales and orders
* Track monthly and daily sales trends
* Identify top-performing cities and states
* Compare veg vs non-veg sales
* Evaluate customer ratings

---

## 📊 Key Metrics (KPIs)

* 💰 Total Sales: $53.01M
* ⭐ Average Rating: 4.34
* 🧾 Avg Order Value: $268.51
* 📦 Total Orders: 197.43K

---

## 📈 Dashboard Features

* Monthly & Daily Sales Trends
* Sales by Food Type (Veg vs Non-Veg)
* State-wise Sales Analysis
* Top 5 Cities by Sales
* Weekly Sales Trend
* Interactive Filters (Month, Category, State)

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Slicers for interactivity
